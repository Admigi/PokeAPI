# PokéAPI

A full-stack Pokédex application built with Spring Boot GraphQL and React.

## Tech Stack

**Backend**
- Java 21 / Spring Boot
- GraphQL
- Maven

**Frontend**
- React 19 / TypeScript
- TanStack Router + Start (SSR)
- Tailwind CSS v4
- Vite

**Infrastructure**
- Docker / Docker Compose

## Features

- Browse all Generation 1 Pokémon in a responsive grid
- Search by name or ID
- Filter by type (OR / AND mode)
- Sort by any stat (HP, Attack, Defense, Sp. Atk, Sp. Def, Speed)
- Pagination
- Detail page with animated stat bars
- Server-side rendering (SSR)

## Getting Started

### Prerequisites
- Docker Desktop

### Run locally
```bash
docker compose up --build
```

- Frontend: http://localhost:3000
- Backend GraphQL: http://localhost:8080/graphql
